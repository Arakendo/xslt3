# Contributing

## This project does not accept external contributions.

`@arakendo/xslt` is **open source** (MIT licensed) but **closed to contributions**.
You are free to:

- Use the code in your own projects (commercial or otherwise) under the MIT License
- Fork the repository
- Modify your fork however you like
- Redistribute under the MIT License terms

You should **not**:

- Open pull requests against this repository — they will be closed automatically
- Open issues requesting features or reporting bugs — the issue tracker is disabled

## Why?

This is a personal project. Keeping contributions closed lets it move at its
own pace without the overhead of review, triage, and community management.

## Found a bug?

If you're using the library and hit a problem, the recommended path is to fork
and fix it in your own copy. You are welcome to publish your fork.

## Security issues

If you discover a security vulnerability, please contact the maintainer
privately rather than filing a public issue or PR.
